Módulo 2: Estructura y elementos

Mapa Conceptual - Función Cuadrática

Función Cuadrática
Definición ▼

Una función cuadrática es una función polinómica de segundo grado que tiene la forma:

f(x) = ax² + bx + c

Donde:

  • a, b, c son coeficientes reales
  • a ≠ 0 (si a = 0, sería una función lineal)
  • x es la variable independiente
Gráfica ▼

La gráfica de una función cuadrática es una parábola:

Si a > 0 → Parábola abre hacia arriba
Si a < 0 → Parábola abre hacia abajo
Ejemplo: f(x) = x² - 4x + 3
a = 1 > 0 → Parábola abre hacia arriba
Elementos ▼
Vértice: Punto máximo o mínimo de la parábola
Eje de simetría: Línea vertical que divide la parábola en dos partes iguales
Raíces/Ceros: Puntos donde la función corta el eje x
Ordenada al origen: Punto donde corta el eje y
Vértice: (h, k) donde h = -b/2a, k = f(h)
Aplicaciones ▼

Las funciones cuadráticas tienen numerosas aplicaciones:

  • Física: Movimiento parabólico
  • Ingeniería: Diseño de estructuras
  • Economía: Optimización de costos y beneficios
  • Arquitectura: Diseño de arcos y puentes
  • Deportes: Trayectoria de proyectiles
Resolución ▼

Métodos para encontrar las raíces:

Fórmula general:
x = [-b ± √(b² - 4ac)] / 2a
Factorización:
Cuando es posible expresar como producto de binomios
Completar el cuadrado:
Reescribir la función en forma vértice
Propiedades ▼
Dominio: Todos los números reales (ℝ)
Rango: Depende del vértice y la concavidad
Es continua en todo su dominio
Tiene un máximo o mínimo absoluto
Es simétrica respecto a su eje
